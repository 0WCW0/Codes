1.软件定位
  学生成绩管理系统，能够实现计算机数据化管理学生成绩及相关人员的信息，能够提供方便快捷的学生信息检索、成绩查询、信息修改等功能，并能够对学生信息、教师信息和管理员信息等使用者进行管理，方便学校管理。
2.运行方式
  安装环境：jdk8 + tomcat8
  启动命令：idea中启动tomcat再启动项目

3.简要说明
  简单的分为：登录、注册、增删查改页面
  学生可以进行查询个人信息，包括个人成绩查询和修改密码的操作
  教师可以进行查询个人信息、学生成绩的录入和查询，个人密码和学生密码的修改的操作
  管理员可以进行对用户（学生和教师）的管理（增删改查），班级的管理（增删改查）。成绩的管理（增删改查）以及学生、教师和本人的密码修改操作
